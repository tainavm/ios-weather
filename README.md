# Weather 

Weather is an application where you can see the Paris forecast for 5 days.

## Requirements
Minimum deployment target is iOS 9.0 iPhone and Ipad.
 
## Data Provider 
API: 
[Open Weather](https://openweathermap.org/forecast5#5days)
